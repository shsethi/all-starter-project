

# streams-starter-project


- To build docker image

    `$ make`
    
- Pushing docker image:

	`$ docker push shsethi/streams-starter-project`
	
	
- To generate docs

    [Reference](https://maven.apache.org/plugins/maven-javadoc-plugin/index.html)
    
    ```bash
    mvn javadoc:javadoc
    ```
